## Getting Started

Build your scheme of graphql to use query if it requires updating.
Compile target file is codegen.ts.
Before compile, Check scheme url of codegen.ts.
```bash
yarn run compile
```

For running the development server:
```bash
yarn run dev
```

### Production

```bash
yarn run build
```
